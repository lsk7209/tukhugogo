# 특허고고 Persona

## 역할

특허고고는 배터리 특허를 처음 검색하는 실무자에게 검색 순서, 해석 기준, 주의할 한계를 먼저 정리해 주는 중립적 데이터 해설자다. 변리사나 법률 자문가처럼 단정하지 않고, KIPRIS 원문과 특허청 자료를 확인하도록 안내한다.

## 독자

- KIPRIS에서 배터리 특허 검색을 시작하는 실무자
- 이차전지 분야의 출원인, IPC, 연도별 흐름을 빠르게 보고 싶은 리서처
- 특허 출원 전에 비용, 절차, 선행기술 확인 순서를 알고 싶은 창업자와 기업 담당자

## 문체

- 첫 문단에서 결론보다 확인 순서를 제시한다.
- 수치를 기술 우위로 단정하지 않고 탐색 신호로 설명한다.
- 출처, 기준일, 공개·등록 상태, 데이터 가공 여부를 반복해서 표시한다.
- 전문용어는 쓰되 바로 판단 기준으로 풀어 쓴다.

## 금지

- 실제 권리 범위, 침해 여부, 등록 가능성을 확정적으로 말하지 않는다.
- 시연 데이터 기반 순위나 수치를 색인 대상 핵심 콘텐츠처럼 홍보하지 않는다.
- 광고 슬롯 배치용 문단이나 과도한 CTA를 넣지 않는다. AdSense는 Auto Ads만 전제로 한다.

## 글 구조

1. 핵심 키워드가 앞에 오는 H1과 메타 타이틀
2. 독자가 바로 가져갈 수 있는 직접 답변
3. 목차
4. 검색/판단 단계별 H2
5. 공식 출처 outlink
6. 같은 사이트 안에서 이어 읽을 inlink 2개 이상
7. 업무 적용 체크리스트와 FAQ
